virtual void lbann::target_layer_distributed_minibatch< T_layout >::setup_data ( )
inlinevirtual

Called by setup(), each layer should override to call its parent and set up the layer's data (e.g. weights). This base method sets up the activations and error signal matrices. This is always called after setup_dims.

If the target layer shares a data reader with an input layer, do not setup the data reader a second time

Note that the data readers for evaluation should not be partitioned over multiple models (otherwise each model will be scored on a different set of data)
