Detailed Description

Data reader for merging multiple data readers. This can take any positive number of data readers, which will be concatenated in the order provided to provide the data, and a single data reader to provide the label. This data reader uses the fetch_datum method of its subsidiary data readers to fetch all data, including the labels.
