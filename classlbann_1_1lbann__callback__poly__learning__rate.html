Detailed Description

Decrease the learning rate by polynomial policy base_lr*(1 - i_cur/i_max)^p, where base_lr is the initial learning rate, i_cur is the current iteration, i_max is the maximum iteration, and p is a parameter.

This constructor takes the policy specific parameters, the exponent (p) and the maximum number of iterations (max_iter). In case that max_iter is set to 0, it is calculated from the number of epochs (n_epochs). n_epochs is not used otherwise.
