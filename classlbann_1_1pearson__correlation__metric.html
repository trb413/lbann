const AbsDistMat &  ground_truth 
)
overrideprotectedvirtual

Computation to evaluate the metric function. This returns the sum of metric values across the mini-batch, not the mean value.

If there is no valid true value and it is the last column of the matrix, the data may not exist due to a mini-batch size imbalance with multiple models. Just skip this element.
